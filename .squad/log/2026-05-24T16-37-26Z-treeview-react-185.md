# Session Log: Tree View React #185 Regression Fix

**Date:** 2026-05-24  
**Topic:** React #185 (Maximum update depth exceeded) when creating response properties after tree view toggle

## Problem

React #185 regression occurred when creating response properties after toggling tree view mode in Custom API Details. Sequence: Tree View → Create Request Parameter → Return to Tree View → Create Response Property → Error.

## Root Cause

Stale `responsePropertyQuery` cache state combined with validation cascades caused re-render loops. GenericTagPicker items instability compounded the issue during remounts.

## Solution

Dallas fixed both sides:
1. **GenericTagPicker**: Made stale-selection clearing idempotent with ref-backed guard
2. **Create Forms**: Memoized picker item arrays to prevent unstable references
